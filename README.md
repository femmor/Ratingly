# Ratingly - A feedback review application

This project was developed using React for the front end and json-server to mock the backend.

### `npm run dev`

Runs the client on Port 3000 and the json-server on Port 5000

### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.\
